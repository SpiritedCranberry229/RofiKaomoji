# RofiKaomoji
A simple rofi script to select, copy and paste you favorite kaomoji! (^Ï‰^)

# Instructions
Before you begin, make sure you have the right dependencies, otherwise you might not be able to correctly see some symbols:

Arch X11: pacman -S xclip 
Arch Wayland: pacman -S wl-clipboard

Arch: pacman -S noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-dejavu
