# RofiKaomoji (*≧ω≦)
A simple rofi script to select, copy and paste you favorite kaomoji! (^ω^)

# Instructions
Before you begin, make sure you have the right dependencies, otherwise you might not be able to correctly see some symbols:

Arch: pacman -S noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-dejavu
Debian/Ubuntu: apt install noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-dejavu
Fedora: dnf install noto-fonts noto-fonts-cjk noto-fonts-emoji ttf-dejavu

### If using X11:
Arch: pacman -S xclip
Debian/Ubuntu: apt install xclip
Fedora: dnf install xclip

### If using Wayland
Arch: pacman -S wl-clipboard
Debian/Ubuntu: apt install wl-clipboard
Fedora: dnf install wl-clipboard
